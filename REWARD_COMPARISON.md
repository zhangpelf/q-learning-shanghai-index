# Q-Learning 奖励函数对比报告

> 2026-07-03 | 上证指数 sh000001 + 有研新材 600206

## 概述

Q-learning 交易策略的核心是**奖励函数** — 它决定了 agent 从每步交易中"学到什么"。不同资产需要不同奖励形态：指数趋势平稳，个股波动剧烈。本文档记录奖励函数迭代的全部结果。

---

## 上证指数 (sh000001)

**数据范围**: 2022-06-23 ~ 2026-07-03 | **seed=42** | 1000 episodes | α=0.01 | γ=0.99

| # | 奖励函数 | 策略收益 | 买入持有 | 超额 | 买卖分布 |
|---|---------|---------|---------|-----|---------|
| 1 | 方向性 ±1 | +16.03% | +21.79% | -5.76% | 171买/230卖/555持 |
| 2 | **幅度×2(无摩擦)** | **+43.94%** | **+21.79%** | **+22.15%** | **125买/387卖/444持** |
| 3 | 不对称×3/×1 | +5.30% | +21.79% | -16.49% | 94买/449卖/413持 |
| 4 | 阈值±0.5% | +14.32% | +21.79% | -7.47% | 92买/193卖/671持 |

### 奖励函数设计详解

每个奖励函数以不同的方式将"次日涨跌"映射为 agent 的学习信号：

**① 方向性 ±1（去漂移）**
```python
买入 = +1  if 明天 ret > 0  else -1
卖出 = +1  if 明天 ret < 0  else -1
持有 = 0
```
只保留涨跌方向，丢弃幅度信息。明天涨 0.1% 和涨 5% 得分完全一样 → agent 在小波动上也频繁买卖，超额 **-5.76%**。

**② 幅度×2（无摩擦）← 最优**
```python
买入 = 明天 ret × 2
卖出 = -明天 ret × 2
持有 = 0
```
"无摩擦" = 持有中性（0 分），交易动作不额外扣手续费。涨跌幅度×2 完整保留信号强度 → agent 只在大涨大跌时出手。超额 **+22.15%**。适用于指数这种波动小、趋势弱的资产。

**③ 不对称（买激进/卖保守）**
```python
买入 = 明天 ret × 8    # 买入放大 8 倍
卖出 = -明天 ret × 3   # 卖出只放大 3 倍
持有 = 0
```
人造买入偏好。指数上没有趋势支撑，偏见导致追涨被套，超额 **-16.49%**。但个股（有研新材）是强趋势股，同样的偏见反而放大了趋势收益（超额 +206%）。

**④ 阈值触发（>0.5%）**
```python
if |明天 ret| > 0.5%:  买入正确 +2 / 错误 -2
if |明天 ret| ≤ 0.5%:  买入正确 +0.5 / 错误 -0.5
持有 = 0
```
连续涨跌幅被一刀切砍成两档。指数 0.3-0.8% 的常用信号区间信息丢失，超额 **-7.47%**。

## 有研新材 (600206)

**数据范围**: 2022-01-04 ~ 2026-07-03 | **seed=42** | 1000 episodes | α=0.01 | γ=0.99

| # | 奖励函数 | 策略收益 | 买入持有 | 超额 | 买卖分布 |
|---|---------|---------|---------|-----|---------|
| 1 | 方向性 ±1 | +296.00% | +242.80% | +53.20% | 253买/375卖/339持 |
| 2 | 幅度×2(无摩擦) | +89.81% | +242.80% | -152.99% | 218买/399卖/349持 |
| 3 | **不对称×3/×1** | **+448.52%** | **+242.80%** | **+205.72%** | **311买/388卖/368持** |
| 4 | 不对称×1/×3 | +99.22% | +242.80% | -143.58% | 2买/559卖/405持 |
| 5 | 阈值±0.5% | +63.92% | +242.80% | -178.88% | 128买/258卖/580持 |

**结论**: **不对称(买入×3/卖出×1)** 大幅跑赢，超额 +206%。原因：
- 有研新材是**趋势股** (2024-2026 涨幅 ~243%)，必须敢买敢持
- 买入奖励×3 vs 卖出×1 → agent 在上涨时追入更快，回调时卖出更犹豫
- 买卖分布 311/388/368 — 买入次数最多 (vs 其他奖励的 2~253)，符合强趋势特征
- 对称或买入惩罚过重的奖励 (#2, #4, #5) 都大幅跑输持有

## 跨资产对比

| 维度 | 上证指数 | 有研新材 |
|------|---------|---------|
| 最优奖励 | 幅度×2(无摩擦) | 不对称×3/×1 |
| 年化超额 | +22.15% | +205.72% |
| 最佳买卖比 | 0.32 (卖为主) | 0.80 (买为主) |
| 资产特征 | 震荡→慢牛 | 强趋势股 |
| 核心原理 | 降低噪音交易频率 | 放大趋势收益 |

**统一规则**: 没有一条奖励函数能同时适配两种资产。奖励设计必须匹配资产的**趋势结构**：
- 震荡/慢牛市 → 无摩擦、中性基准 → 减少假信号
- 强趋势股 → 不对称激励 → 拥抱趋势

## 后续方向

1. **多种子稳定性**: 当前 seed=42，需 5-10 个种子验证超额可持续性
2. **在线学习**: 每日增量更新 Q 表，适应市场风格漂移
3. **组合层**: 将两信号合成一个仓位决策 (如: 指数卖出则个股减半)
4. **状态细化**: 当前 1% bin 粒度太粗，可测试 0.5% 或自适应分桶
